# 参数平均即逐层蒸馏：联邦学习中知识传递机制的再思考

## 1. 动机

联邦学习（Federated Learning, FL）已有大量工作证明了其收敛性，但以下更本质的问题仍缺乏清晰的解释：

- **为什么参数平均可以在模型之间传递知识？** 神经网络具有高度非线性，但线性的参数平均却能有效传递"模型知识"，其机制并不显然。
- **如何更合理地度量模型"差异"并设计更鲁棒的聚合方案？** 现有工作多采用参数的 2-范数度量模型差异，但这是否是最本质的度量方式？

本文提出一个统一视角来回答上述2个问题：**参数平均的本质是数据无关的逐层蒸馏**。基于该视角，我们可以自然地理解参数平均的有效性与局限性，并推导出面向数据异质性的鲁棒聚合方案。

## 2. 核心观点：参数平均是数据无关的逐层蒸馏

### 2.1 数学推导

考虑单层线性网络（忽略 bias），N 个客户端参数分别为 $W\_1, \cdots, W\_N$，输入为 $x$。若要将 N 个模型蒸馏到一个模型 $W$ 中，优化目标为：

$$\min\_W \sum\_{i=1}^{N} |Wx - W\_i x|\_2^2 \quad (1)$$

对 $W$ 求导可得最优解为：

$$W = \frac{1}{N} \sum\_{i=1}^{N} W\_i + Z, \quad \text{其中 } Zx = 0$$

**关键发现：** 无论输入 $x$ 取何值，$W = \frac{1}{N}\sum\_{i=1}^{N} W\_i$（即参数平均）都是最优解。这意味着：**对于单层线性层，参数平均在任何数据分布下都是最优的蒸馏方案。**

### 2.2 从单层到多层：逐层蒸馏的直觉

现代神经网络是分层的，每层大多是线性矩阵乘法操作（卷积层、全连接层），由非线性操作串联（激活函数、attention 等）。在这一结构下，FL 的参数平均操作可以被等价地理解为：

1. 将 N 个客户端模型的第 $l$ 层权重分别拆出；
2. 对第 $l$ 层，按公式 (1) 的优化目标蒸馏得到全局模型的第 $l$ 层权重；
3. 对所有层重复上述过程，拼接各层得到完整的全局模型。

即：**参数平均 = 数据无关的逐层蒸馏**。

## 3. 面向数据异质性的改进：鲁棒聚合方案

### 3.1 问题：平方距离对离群值的敏感性

即使在"同输入"假设近似成立的场景下，公式 (1) 中的平方距离度量也存在缺陷。在数据异质性场景下，少数客户端的 feature $W\_i x$ 可能与多数客户端差距很大，平方距离会使这些离群客户端对聚合结果产生不成比例的影响，拉偏全局模型。

一个自然的改进思路是：将逐层蒸馏中的平方距离替换为对离群值更鲁棒的距离度量。

### 3.2 广义距离度量与不动点迭代

将公式 (1) 中的 2 次方推广为 $\alpha$ 次方，定义广义聚合目标：

$$W^\* = \arg\min\_W \sum\_{i=1}^{N} |W - W\_i|^{\alpha}\_2 \quad (2)$$

对 $W$ 求导并令梯度为零，可得到如下不动点迭代格式：

$$W\_{t+1} = \frac{\sum\_{i=1}^{N} |W\_i - W\_t|\_2^{\alpha-2} \cdot W\_i}{\sum\_{i=1}^{N} |W\_i - W\_t|\_2^{\alpha-2}}$$

该迭代具有清晰的物理含义——每轮用与当前估计的距离对各客户端参数进行加权平均，距离的幂次由 $\alpha$ 控制：

- $\alpha = 2$：所有权重相等，退化为普通参数平均（FedAvg）；
- $\alpha = 1$：即经典的 Weiszfeld 迭代，求解几何中位数，离当前估计越远的客户端权重越小，天然抑制离群值；
- $1 < \alpha < 2$：在 FedAvg 与几何中位数之间提供平滑的插值，可根据数据异质性程度灵活调节鲁棒性。

## 5. 实验计划

**目标：** 评估广义距离度量（$\alpha < 2$）在不同数据异质性场景下的有效性。

### 5.1 $\alpha$ 灵敏度分析

在标准 FL benchmark（CIFAR-10/100、FEMNIST）上，固定 non-IID 设置（如 Dirichlet $\beta=0.1$），扫描 $\alpha \in {1.0, 1.2, 1.5, 1.8, 2.0}$，记录全局模型的测试精度和收敛速度。重点与 FedAvg（$\alpha=2$）和几何中位数（$\alpha=1$）进行对比，观察中间 $\alpha$ 值是否能取得更好的平衡。

在 IID 分布的情况下，也需要做基础的验证。看看能否提升各个客户端随机偏移带来的影响。

### 5.2 异质性程度与最优 $\alpha$ 的交互

在不同 non-IID 程度（$\beta \in {0.1, 0.5, 1.0, 5.0}$）下分别测试各 $\alpha$ 值，验证核心假设：**数据异质性越强，最优 $\alpha$ 越小**（即需要更强的鲁棒性）。若假设成立，可进一步探索 $\alpha$ 的自适应选择策略。

### 5.3 不动点迭代收敛性

记录不同 $\alpha$ 下不动点迭代的收敛轮次和数值稳定性，评估算法的实际可用性。重点关注 $\alpha$ 接近 1 时迭代是否出现振荡或收敛缓慢的问题。

### 5.4 实验设置概览

| 维度    | 设置                                                                           |
| ----- | ---------------------------------------------------------------------------- |
| 数据集   | CIFAR-10, CIFAR-100                                                          |
| 模型    | ResNet-18, 简单 CNN                                                    |
| 客户端数量 | 10, 50, 100                                                                  |
| 数据划分  | IID 分布，Dirichlet 分布，$\beta \in {0.1, 0.5, 1.0, 5.0}$                            |
| 聚合方法  | FedAvg ($\alpha=2$), 几何中位数 ($\alpha=1$), 广义聚合 ($\alpha \in {1.2, 1.5, 1.8}$) |
| 对比方法  | FedMedian, FedProx                                                           |
| 评估指标  | 全局模型测试精度、收敛轮次、不动点迭代轮次                                                        |

## 6. 总结

| 视角      | 核心结论                                       |
| ------- | ------------------------------------------ |
| 参数平均的本质 | 数据无关的逐层蒸馏，对单层线性网络是最优的                      |
| 改进方向    | 使用对离群值更鲁棒的广义距离度量（$\alpha < 2$），通过不动点迭代求解   |
| 待验证     | 广义聚合在不同异质性场景下的最优 $\alpha$ 选择及其与现有鲁棒聚合方法的对比 |
